scipy.linalg.lu_solve¶

scipy.linalg.lu_solve((lu, piv), b, trans=0, overwrite_b=False)¶

Solve an equation system, a x = b, given the LU factorization of a

Parameters :

(lu, piv) :

b : array

Right-hand side

trans : {0, 1, 2}

Type of system to solve:

trans

1

2

x : array
